ActiveRecordLite

GitHub
  • ORM library for Ruby based on Ruby on Rails’ ActiveRecord
  • Instantiate new Ruby objects based on search conditions
  • Define one-to-many, one-to-one, and many-to-many associations between models
  • Execute CRUD actions on records in the database

ActiveRecordLite is a recreation of ActiveRecord’s core features using SQLite3. Ruby’s metaprogramming combined with a simple DBConnection class for executing SQL queries make it easy to build a flexible and intuitive API much like ActiveRecord’s.
